##  Features

### 1. User Registration
- Clean, responsive signup form  
- Live validation:
  - Phone number must be 10 digits  
  - PIN code must be 6 digits  
  - Email must be valid  
- Green border = correct, red border = error  
- Custom success popup with progress bar  
- Auto-navigation to the Destinations page
